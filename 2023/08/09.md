+ Rubyのメソッド名で?で終わるメソッドは慣習として真偽値(trueかfalse)を返す。
このようなメソッドを述語メソッドという
+ Ruby3.0ではendを省略して1行でメソッドを定義できる(エンドレスメソッド定義)
```rb
# 通常のメソッド定義
def add(a, b)
    a + b
end

# エンドレスメソッド定義
def add(a, b) = a + b

# 呼び出し方はどちらも同じ
add(1, 2) #=> 3
```
