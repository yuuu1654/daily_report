# わかったこと 
+ HTTPリクエスト
    + リクエストは3要素で構成される
    + ①リクエストライン
        + メソッド
        + リクエストURI
        + HTTPバージョン
    + ②ヘッダー
        + サーバーに対する追加情報を送信
        + リクエストラインに続くさまざまなメタデータが含まれる
        + Content-Typeには、送信データのフォーマットが入る
        + User-Agentには、クライアントのブラウザ情報が入る
        + Cookieもヘッダーの中に含まれる
    + ③ボディ
        + Content-Typeの内容に従って、様々なデータが入る
        + prefから始まる
